# Server Performance Stats (Beginner)

This is a beginner-level DevOps project to analyze basic server performance stats using Bash scripting.

## Features

- Total CPU usage
- Total memory usage (Free vs Used including percentage)
- Total disk usage (Free vs Used including percentage)
- Top 5 processes by CPU usage
- Top 5 processes by memory usage
- Additional info (OS version, uptime, load average, logged in users, failed login attempts)

## How to Run

1. Clone or download the repository.
2. Make the script executable:
   ```bash
   chmod +x server-stats.sh
3. Execute the script
    ```bash
    ./server-stats.sh
